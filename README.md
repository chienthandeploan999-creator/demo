# Aurora Nails (Gold & Black)

Website tĩnh cho salon nail cao cấp phục vụ thị trường Mỹ.

- Màu chủ đạo: vàng kim (#d4af37) và đen huyền bí (#0b0b0e)
- Đối tượng: Người lớn (làm móng, làm tay, chăm sóc da, chăm sóc móng) và Trẻ em (làm móng, dưỡng móng, dưỡng da mặt)

## Cấu trúc
- `index.html`: Trang chính và các section
- `styles.css`: Giao diện vàng đen, responsive
- `script.js`: Tabs, điều hướng, xử lý form đặt lịch
- `assets/favicon.svg`: Favicon

## Chạy dự án
Mở file `index.html` trong trình duyệt. Không cần backend.

## Tuỳ biến nhanh
- Cập nhật thông tin liên hệ và giờ mở cửa trong phần footer của `index.html`.
- Sửa bảng giá trong section `#bang-gia`.
- Ảnh thư viện: đặt ảnh ngay cạnh `index.html` với tên `1.jpg` … `6.jpg` (hoặc sửa mảng `GALLERY_IMAGES`).
- Icon PWA: thêm `icon-192.png` và `icon-512.png` ở thư mục gốc để icon đẹp trên điện thoại.
